Digital Connect Installation Guide

Requirements

  • XAMPP
  • Visual Studio Code
  • Web Browser
  • MySQL Database

Installation Steps

  1. Download or clone the project repository.
  2. Extract the ZIP file.
  3. Open the project folder in Visual Studio Code.
  4. Start Apache and MySQL in XAMPP.
  5. Open phpMyAdmin and create a database called digital_connect.
  6. Import the SQL file into the database.
  7. Move the project folder to the htdocs folder in XAMPP.
  8. Open your browser and go to:

    9. http://localhost/Digital-Connect/

  9. Register or log in to use the website.
